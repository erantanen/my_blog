Okay, everybody loves animation here is a simple ball moving in a diagonal pattern across the canvas. Using some simple addition a quick mod to x & y co-ordinates can be made.
circle animation

`` dx += 10; dy += 10;``

The first silly thing I learned was that the timing is not what makes the animation, but the ability to clear the canvas before the next draw. We do that with
function clear() {   ctx.clearRect(0, 0, width, height); }

Slowing the animation down is actually quite easy when adjusting setInterval right now it is at 20 but if you increase it say by 5 to 25 you will see a significant difference. Now that the myCircle is a function it can be re-used, the same for clear and init.

Now lets make it a bit more complicated!
2 rectangles animated
I have created another function for rectangles.
function myRect(x,y,w,h,c){  ctx.fillStyle=c;  ctx.fillRect(x,y,w,h);  ctx.stroke(); }
And if you notice the c for input, it is for color.
var c = "#000080";
